STUDENT API

This is a  simple rest API i have built with Node.Js and Express to manage student data.
This project demonstrates CRUD operations, query filtering, and basic validation.

---

These are the features 

- Get all students
- Get a single student by ID
- Create a new student
- Update an existing student
- Delete a student
- Filter students by:
  - year
  - status
  - name (case-insensitive)

---

  Tech stack

- Node.js
- Express.js
- JavaScript
- Thunder Client / Postman (for testing)
- Git & GitHub
